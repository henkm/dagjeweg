# Dagjeweg

Met deze gem kun je eenvoudig data opvragen via de DagjeWeg.NL API.

## Installatie

Voeg deze regel toe aan je Gemfile:

    gem 'dagjeweg'

En voer uit:

    $ bundle

Of installeer het zelf:

    $ gem install dagjeweg
